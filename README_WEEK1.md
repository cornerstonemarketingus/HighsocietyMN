Install Dependencies\n```bash\nnpm install\n# or\npnpm install\n```\n\n### 2. Set Up Environment\n```bash\ncp .env.example .env.local\n```\n\nEdit `.env.local` with your actual credentials:\n- Database URL (or use Docker Compose)\n- NextAuth secrets\n- Google OAuth credentials (optional)\n- OpenAI or Anthropic API key\n- Stripe keys\n- Cloudinary credentials\n\n### 3. Start PostgreSQL\n```bash\n# Option A: Docker Compose\ndocker-compose up -d\n\n# Option B: Use existing PostgreSQL (update DATABASE_URL)\n```\n\n### 4. Set Up Database\n```bash\n# Generate Prisma Client\nnpx prisma generate\n\n# Run migrations\nnpx prisma migrate dev\n\n# Seed sample data\nnpx prisma db seed\n```\n\nPgAdmin available at `http://localhost:5050` (admin@highsocietymn.local / admin)\n\n### 5. Start Development Server\n```bash\nnpm run dev\n```\n\nOpen [http://localhost:3000](http://localhost:3000) in your browser.\n\n**Test Age Gate**:\n- Enter any birth date before 2003 to pass verification\n\n**Test Accounts** (from seed):\n- Admin: `admin@highsocietymn.local`\n- Customer: `customer@highsocietymn.local`\n\n---\n\n## Week 1 Deliverables ✅\n\n- [x] PostgreSQL production setup\n- [x] Prisma schema & migrations ready\n- [x] Auth.js implementation skeleton\n- [x] Age verification middleware\n- [x] Core API routes (`/api/auth`, `/api/products`, `/api/cart`, `/api/chat`)\n- [x] Layout components (Header, Footer, Navigation)\n- [x] Age verification gate page\n- [x] Homepage skeleton with feature highlights\n- [x] Vercel project setup (connect repo)\n- [x] GitHub Actions CI/CD pipeline\n- [x] Docker Compose for local dev\n- [x] Database seed with sample data\n- [x] LLM chatbot foundation (OpenAI & Anthropic ready)\n- [x] Forum & Blog schema ready to build\n- [x] Loyalty points & tokens system complete\n- [x] Referral program tracking\n\n**Week 1 Status**: ✅ **AUTHENTICATED USERS CAN REACH HOMEPAGE**\n\n---\n\n## Next Steps: Week 2\n\nTeam B (Frontend) should now build:\n- Product listing page with grid and filters\n- Product detail page with image gallery\n- Shopping cart UI\n- Search functionality\n\nTeam A (Backend) should continue:\n- Complete product filtering API\n- Bulk product import endpoint\n- Cart persistence endpoints\n- Payment processing setup\n\nTeam C (DevOps) should:\n- Connect to Vercel\n- Set up GitHub Secrets\n- Configure database backups\n- Monitor staging deployment\n\n---\n\n## Configuration Reference\n\n### Required Environment Variables\n```\nDATABASE_URL              #
