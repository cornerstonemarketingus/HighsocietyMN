# High Society MN - Vercel Deployment Guide\n\n## ✅ You've Deployed! Here's What's Next\n\n### 1. **Add PostgreSQL Database**\n\n**Option A: Neon (Recommended - Free)**\n```\n1. Go to https://neon.tech\n2. Sign up with GitHub\n3. Create a new project\n4. Copy the connection string\n5. In Vercel Dashboard:\n   - Go to Settings → Environment Variables\n   - Add DATABASE_URL = [your neon connection string]\n   - Redeploy\n```\n\n**Option B: Railway.app (Free tier)**\n```\n1. Go to https://railway.app\n2. New Project → PostgreSQL\n3. Copy DATABASE_URL\n4. Add to Vercel environment variables\n```\n\n**Option C: AWS RDS (Paid)**\n```\n1. AWS Console → RDS\n2. Create PostgreSQL instance\n3. Get connection string\n4. Add to Vercel\n```\n\n---\n\n### 2. **Configure Environment Variables in Vercel**\n\nGo to: **Vercel Dashboard → Project Settings → Environment Variables**\n\nAdd these variables:\n\n```\n# Database\nDATABASE_URL=postgresql://user:password@host:5432/highsocietymn\n\n# NextAuth\nNEXTAUTH_URL=https://your-deployment.vercel.app\nNEXTAUTH_SECRET=generate-with: node -e \"console.log(require('crypto').randomBytes(32).toString('hex'))\"\n\n# Google OAuth (Optional, for sign-in)\nNEXTAUTH_PROVIDERS_GOOGLE_ID=your-google-client-id\nNEXTAUTH_PROVIDERS_GOOGLE_SECRET=your-google-secret\n\n# LLM Chatbot\nLLM_PROVIDER=openai\nOPENAI_API_KEY=sk-your-openai-key\n\n# Payments\nSTRIPE_SECRET_KEY=sk_test_your-stripe-key\nNEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=pk_test_your-stripe-key\n\n# File Storage\nNEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=your-cloud-name\nCLOUDINARY_API_KEY=your-api-key\nCLOUDINARY_API_SECRET=your-api-secret\n```\n\n---\n\n### 3. **Run Database Migrations on Vercel**\n\nOnce DATABASE_URL is set, run migrations:\n\n```bash\n# Connect to Vercel via CLI (requires Vercel CLI installed)\nvercel env pull\n\n# Run Prisma migrations\nnpx prisma migrate deploy\n\n# Seed database with test data\nnpx prisma db seed\n```\n\nOr use Vercel's CLI:\n```bash\nnpm i -g vercel\nvercel login\nvercel env pull .env.local\nnpx prisma migrate deploy\n```\n\n---\n\n### 4. **Your Live URL**\n\nAfter deployment, Vercel provides:\n- **Live URL**: `https://highsocietymn-[random-id].vercel.app`\n- **Check Status**: Deployments tab in Vercel Dashboard\n\n---\n\n### 5. **Test the Live Site**\n\n1. **Visit**: `https://your-deployment.vercel.app`\n2. **Age Gate**: Enter birth date before 2003\n3. **Homepage**: See features & CTA\n4. **Products API**: `https://your-deployment.vercel.app/api/products`\n5. **Chat API**: `https://your-deployment.vercel.app/api/chat` (authenticated)\n\n---\n\n### 6. **Custom Domain Setup**\n\n1. In Vercel Dashboard → Settings → Domains\n2. Add custom domain: `highsocietymn.com` (or your domain)\n3. Update DNS with your registrar\n4. CNAME/A record pointing to Vercel\n\n---\n\n### 7. **Monitor & Debug**\n\n**View Logs**:\n```bash\nvercel logs\n```\n\n**Check Build Status**: Vercel Dashboard → Deployments\n\n**Common Issues**:\n- ❌ `DATABASE_URL not set` → Add environment variable & redeploy\n- ❌ `Build fails` → Check `.github/workflows/ci.yml` logs\n- ❌ `Age gate not working` → Check localStorage in browser DevTools\n\n---\n\n### 8. **Next: Connect Your Team**\n\n```bash\n# Share Vercel project with team\n1.
